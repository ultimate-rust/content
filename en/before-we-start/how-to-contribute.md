

# How to Contribute

## Support with a Subscription

The most direct way to keep Ultimate Rust alive is through a subscription. It grants access to exercises throughout the lessons and, more than that, ensures this project continues to have the time and resources to grow.

If you've already read about how the project works and want to support it, <a>click here to see the plans.</a>

## Be Part of the Building

Ultimate Rust is yours as much as it is mine.

Yes, I was the one who wrote the first words, who organized the chapters, who chose the examples. But from the moment you start reading it, it becomes yours too. And if you want, you can leave your mark on it.

Maybe you'll find a typo that slipped past me. Maybe an explanation that could be clearer. Maybe an example you created that illustrates a concept better. Or maybe you simply have an idea for how to make this book even more didactic, even more complete, even more useful for those who are learning.

Don't ignore that impulse.

This book is open-source precisely because I believe the best way to teach Rust is by doing it together, as a community. Every person who contributes brings a unique perspective, a different experience, a point of view that I alone could never have.

If you want to be part of this construction, if you want to make this book better not just for yourself but for everyone who comes after, I invite you to contribute. It doesn't matter if it's something small or large. It doesn't matter if you're a beginner or experienced. What matters is that you care.

Visit the [Contribution Guide](/en/contribution-guide) page and see how simple it is to be part of this project. Together, we can create the best Rust resource in English *(and possibly globally)* that has ever existed.
