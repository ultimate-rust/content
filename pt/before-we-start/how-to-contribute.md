# Como Contribuir

## Apoie com uma assinatura

A forma mais direta de manter o Ultimate Rust vivo é através de uma assinatura. Ela garante acesso aos exercícios ao longo das aulas e, mais do que isso, garante que este projeto continue tendo tempo e recursos para crescer.

Se você já leu sobre como o projeto funciona e quer apoiar, <a>clique aqui para conhecer os planos.</a>

## Faça Parte da Construção

O Ultimate Rust é seu tanto quanto é meu.

Sim, fui eu quem escreveu as primeiras palavras, quem organizou os capítulos, quem escolheu os exemplos. Mas a partir do momento em que você começar a lê-lo, ele também passa a ser seu. E se você quiser, pode deixar sua marca nele.

Talvez você encontre um erro de digitação que me escapou. Talvez uma explicação que poderia ser mais clara. Talvez um exemplo que você criou e que ilustra melhor um conceito. Ou talvez você simplesmente tenha uma ideia de como tornar este livro ainda mais didático, ainda mais completo, ainda mais útil para quem está aprendendo.

Não ignore esse impulso.

Este livro é open-source justamente porque acredito que a melhor forma de ensinar Rust é fazendo isso juntos, como comunidade. Cada pessoa que contribui traz uma perspectiva única, uma experiência diferente, um olhar que eu sozinho jamais teria.

Se você quer fazer parte desta construção, se quer deixar este livro melhor não apenas para você, mas para todos que virão depois, eu te convido a contribuir. Não importa se é algo pequeno ou grande. Não importa se você é iniciante ou experiente. O que importa é que você se importa.

Acesse a página [Guia de Contribuição](../contribution-guide.md) e veja como é simples fazer parte deste projeto. Juntos, podemos criar o melhor recurso de Rust em português *(e possívelmente global)* que já existiu.
